Parameters

testTypeEnum value for specifying testType

NOTE: There are two types of testType

  1. SDL_STC_TEST-> FOR THIS TEST STATUS SHOULD BE PASS.
  2. SDL_STC_NEG_TEST-> FOR THIS TEST STATUS SHOULD BE FAIL.
